# **App Name**: SonicFlow

## Core Features:

- Online Music Search: Search for music by pulling resources, songs, and lyrics from Google, YouTube, and Spotify. Utilize an AI tool to filter resources based on relevance and legality, to prioritize stable data sources and minimize potentially copyrighted data
- Music Playback: Play selected songs with standard playback controls (play, pause, skip, repeat, shuffle).
- Lyrics Display: Display song lyrics in real-time, synchronized with the music playback. Display these lyrics over a frosted glass material effect, and ensure lyrics auto-scroll and highlight to match the currently playing section.
- UI Customization: Allow users to customize the player's UI to suit their style (e.g., themes, color schemes, layout options). Store the setting in local storage
- Background Album Art: Use the album cover of the currently playing song as the background for the music player screen.
- Equalizer: Implement an equalizer feature with different sound profiles that really work, allowing users to adjust audio frequencies to their preference.
- Output Device Selection: Add an icon where users can choose the playback device, such as earpods, phone speaker, or Bluetooth device.

## Style Guidelines:

- Primary color: Vibrant green (#1DB954), reminiscent of Spotify, evoking energy and rhythm.
- Background color: Dark gray (#121212), for a modern and immersive experience.
- Accent color: Light green (#B3FFCC), providing a subtle contrast to the primary color.
- Body and headline font: 'Poppins' sans-serif, giving it a precise and fashionable style.
- Use simple, minimalist icons for playback controls and other functions. Icon style to use filled outline to enhance visual appeal and make it clean.
- The layout mimics elements from the attached screenshots, with a dark interface, frosted glass effects and album art backgrounds.
- Employ subtle animations when music starts/stops, and when transitioning between tracks.
